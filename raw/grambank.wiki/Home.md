This is the Grambank wiki. It is used to store documentation of features, coding methodology, terminology and other information for coders and users.  

<img align="right" src="https://github.com/user-attachments/assets/17d8b789-58c5-4135-aec3-ecb7ec74cdc7" data-canonical-src="https://github.com/user-attachments/assets/17d8b789-58c5-4135-aec3-ecb7ec74cdc7" width="350" height="400" />

The wiki contains the following types of documentation of the project:

1. Background information about the questionnaire
2. Practical information for users and collaborators
3. Notes on coding methodology
4. Notes on the linguistic terminology and comparative concepts we adopt
5. Wiki articles about the individual Grambank features
6. Practical guides for coders

You can find [a list of all articles for each feature here](https://github.com/grambank/Grambank/wiki/List-of-all-features). If you are a coder looking for new tasks, there is a priority list of in our shared Google Spreadsheet that includes both new languages and revision tasks.

If you are not a project member but want to contribute, [please see this page](https://github.com/grambank/grambank/wiki/Contribute) and the [FAQ](https://github.com/grambank/grambank/wiki/FAQ-(general)).
